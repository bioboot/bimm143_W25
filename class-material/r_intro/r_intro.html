Functions

Being productive in R is all about using functions - we call functions when we read a dataset, perform a statistical analysis, make a fancy visualization or do just about anything else.


All functions are accessed like so:


functionName(arg1 = val1, arg2 = val2, and so on)


Notice that we use the = sign here for setting function inputs (a.k.a. setting function ‚Äúarguments‚Äù) and that each one is separated by a comma.

The previous seq(1, 10) example above also demonstrates something about how R resolves function arguments.

Vectors

Vectors are the most fundamental data structures in R. Their job is to hold a contiguous collection of ‚Äúelements‚Äù of the same ‚Äútype‚Äù.


For example a set of numbers (what we call numeric type), words (character type) or True and False vales (logical type).

To create vectors we combine values with the function c():

Vectorization

Vectors are the basis of one of R‚Äôs most important features: vectorization. Vectorization allows us to loop over all elements in a vector without the need to write an explicit loop.

Data.frames combine vectors into tables

A data.frame is a table (or two-dimensional array-like structure) where columns can contain different types of data (numeric, character, logical, etc.).


Data.frames are ideal for managing and analyzing datasets with rows and columns like we find in spreadsheets.

Errors, warnings, and messages

R will show red text in the console pane in three different situations:

  • Errors: When the red text is a legitimate error, it will be prefaced with ‚ÄúError in‚Ä¶‚Äù and will try to explain what went wrong. Generally when there‚Äôs an error, the code will not run.

  • Warnings: When the red text is a warning, it will be prefaced with ‚ÄúWarning:‚Äô and R will try to explain why there‚Äôs a warning. Generally your code will still work, but with some caveats.

  • Messages: When the red text doesn‚Äôt start with either ‚ÄúError‚Äù or ‚ÄúWarning‚Äù, it‚Äôs just a friendly message. You‚Äôll see these messages when you load R packages for example.

üîë Key point:

R reports errors, warnings, and messages in a glaring red font, which makes it seem like R is scolding you.


Don‚Äôt panic, it doesn‚Äôt necessarily mean anything is wrong and indeed it is not always a bad thing. However, you should read this text to make sure things are as you expect.


Over time we will learn more ‚ÄúR speak‚Äù and get more comfortable generally working in the console. Again Google and ChatGPT can help you interpret these important messages.
